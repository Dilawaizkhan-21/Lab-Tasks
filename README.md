# Lab-Tasks
OOP Lab Tasks and Programming Experiments in C++

# Week 2:

**Experiment 1**

**Title:** Student Record Management Using Structure

**Objective:**
To store and display student information using structures in C++.

**Description:**
This program uses a Student structure to store a student's name, roll number, and marks of three subjects. It calculates the total marks and displays all information.

**Concepts Used:**
• Structures
• User Input/Output
• Arithmetic Operations

**Experiment 2:**

**Title:** Date Validation Using Structure

**Objective:** To validate and display a date entered by the user using structures in C++.

**Description: **This program uses a Date structure to store day, month, and year. It validates the entered date by checking the month range, calculating maximum days for each month including leap year logic for February, and verifying the day. The program keeps prompting until a valid date is entered.

**Concepts Used:**

Structures • Input Validation • Loops • Conditional Logic • Leap Year Calculation

**Experiment 3:**

**Title:** Employee Record Management Using Structure

**Objective:** To store and display employee records while preventing duplicate IDs using structures in C++.

**Description:** This program uses an Employee structure to store employee number and salary for 3 employees. It checks for duplicate employee numbers before accepting input and displays all records using a display function.

**Concepts Used:**

Structures • Arrays of Structures • Nested Loops • Duplicate Detection • User Input/Output

# Week 3

**Experiment 1:**

**Title:** Employee Class with Constructors

**Objective:** To demonstrate constructor overloading using an Employee class in C++.

**Description:** This program defines an Employee class with name, ID, department, and position. It implements three constructors — default, with all values, and with name and ID only — along with getters and setters for each attribute.

**Concepts Used:**

Classes and Objects • Constructor Overloading • Encapsulation • Getters and Setters

# Week 5

**Experiment 1**

**Title:** Payroll System Using a Class

**Objective**: To calculate and display weekly payroll for 7 employees using a Payroll class in C++.

**Description:** This program defines a Payroll class that stores hourly pay rate, hours worked, and total pay. The setHoursWorked() method validates that hours are between 0 and 60, re-prompting on invalid input, and total pay is calculated automatically inside the setter. An array of 7 Payroll objects is used to collect data and display a formatted weekly payroll report.

**Concepts Used:**

Classes and Objects • Input Validation in Setter • Arrays of Objects • Formatted Output (iomanip) • Encapsulation

# Week 8

**Experiment 1**

**Title**: Publication Hierarchy Using Inheritance

**Objective:** To demonstrate single inheritance by deriving Book and Tape classes from a Publication base class in C++.

**Description**: This program defines a Publication base class with title and price. Book adds page count and Tape adds playing time. Both derived classes override getdata() and putdata() to extend the base class functionality.

**Concepts Used:**

Inheritance • Method Overriding • Protected Members • Scope Resolution Operator
